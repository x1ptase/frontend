#### 1. Toán tử số học - Arithmetic
```javascript
var a = 1 + 2;
console.log(a;)
```
```javascript
+   --> Cộng
-   --> Trừ
*   --> Nhân
**  --> Lũy thừa
/   --> Chia
%   --> Chia lấy số dư
++  --> Tăng 1 giá trị số
--  --> Giảm 1 giá trị số
```
